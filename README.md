# Rain Audio

This contains rain audio files, a Python script to play them interactively, and a text script with lines for each audio.  

Files included:  
- play_audio.py  
- sample.txt  
- rain_audio1.mp3, rain_audio2.mp3, rain_audio3.mp3, rain_audio4.mp3, rain_audio5.mp3  

Run play_audio.py in Python to play the audios.
